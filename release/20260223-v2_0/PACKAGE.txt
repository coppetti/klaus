HYBRID MEMORY:
==============
The Hybrid Memory system uses:
- SQLite: Fast writes, simple queries, reliable fallback
- Kuzu Graph: Semantic relationships, context chains, topic clusters

All three interfaces (IDE, Web, Telegram) share the same
intelligent memory with relationship awareness.
