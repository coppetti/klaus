# Klaus Workspace

This is your personal workspace for Klaus.

## Structure

- `memory/` - SQLite and Kuzu graph databases for hybrid memory
- `cognitive_memory/` - Episodic and knowledge graph storage
- `semantic_memory/` - Semantic memory storage
- `projects/` - Your project files
- `uploads/` - File uploads from Web UI
- `web_ui_data/` - Session data and settings

## Getting Started

1. Copy `.env.example` to `.env` and add your API keys
2. Run `./setup.sh` to initialize
3. Start with `./scripts/start-services.sh`
